Privacy Controls
Hide Profile

Hiding your profile will make sure it is not publically visible and will not be returned in search results.

 Export Profile

Exporting your profile will download a data file (json format) containing all your stored profile data and uploaded profile photos.

 Delete Profile

Deleting your profile will remove all your profile data (including uploaded photos) from Tech Mentors. It will also remove your authentication account with the Tech Mentors authentication provider (Auth0).
